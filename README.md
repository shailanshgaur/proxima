# Proxima

Proxima is a hyperlocal resident services marketplace for apartment societies and gated communities.

## Repo Map

- `src/`: shipping application code
- `src/pages/*New.tsx`: in-progress redesigned resident flows
- `src/lib/`: auth, vendor, booking, review, and storage services
- `src/styles/`: app styling and design tokens used by the new UI
- `docs/product/`: product direction and positioning
- `docs/design/`: design system and visual language
- `docs/testing/`: testing strategy and launch-readiness docs
- `docs/adr/`: architecture and organization decisions
- `archive/zingconnect/`: legacy ZingConnect artifacts preserved for reference
- `design-system/`: exploratory design-system working files
- `scripts/`: deployment and pre-launch operational scripts

## Current Product State

The app foundation is live in code, with a newer Proxima-branded experience being wired in on top of the first functional pass. Legacy ZingConnect source material is intentionally archived rather than deleted so product context stays available without crowding the working surface.
